linux安装mysql5.6

发表于 2019-01-24 | 分类于 -linux环境搭建 | | 阅读次数:

环境
1.操作系统：CentOS 6.8 64位
2.安装版本：mysql-5.6.43-linux-glibc2.12-x86_64.tar.gz
3.下载地址：https://dev.mysql.com/downloads/file/?id=482460
4.所需工具：Xshell6.0 FlashFXP5.4.0

安装步骤

第一步 卸载老版本

查看命令：rpm -qa|grep -i mysql
停止命令：service mysql stop
删除命令：rpm -e --nodeps 包名

第二步 上传并解压

上传到/usr/local目录下
解压：tar -zxvf mysql-5.6.43-linux-glibc2.12-x86_64
重命名：mv mysql-5.6.43-linux-glibc2.12-x86_64 mysql

第三步 创建用户组和用户

检查是否已有：groups mysql
创建mysql用户组：groupadd mysql
创建mysql用户：useradd -r -g mysql mysql

第四步 更新权限并安装

进入mysql目录：cd /usr/local/mysql
修改权限：chown -R mysql:mysql ./
执行安装脚本：./scripts/mysql_install_db --user=mysql
修改当前目录拥有者为root用户：chown -R root:root ./
修改data目录拥有者为mysql：chown -R mysql:mysql data

第四步 修改密码及配置

启动mysql：./support-files/mysql.server start
更改密码：./bin/mysqladmin -u root -h localhost.localdomain password 'root'
登录mysql：./bin/mysql -h127.0.0.1 -uroot -proot
登录之后修改其他用户密码：update mysql.user set password=password('root') where user='root'; flush privileges;
增加远程访问：grant all privileges on *.* to root@'%' identified by 'root'; flush privileges;

第五步 将MySQL加入Service系统服务

复制启动脚本：cp support-files/mysql.server /etc/init.d/mysqld
添加系统服务：chkconfig --add mysqld 
添加开机启动：chkconfig mysqld on
配置环境变量：export PATH=$PATH:/usr/local/mysql/bin
配置my.cnf添加：
character_set_server=utf8 #设置字符集utf8
lower_case_table_names=1 #表名存储在磁盘是小写的，但是比较的时候是不区分大小写
max_allowed_packet=100M #限制Server接受的数据包大小

最后 启动/停止/重启命令

service mysql start
service mysql stop
service mysql restart
